# Example artifacts

These files are **synthetic examples** generated from the real `SkillFactoryState` implementation.

They exist to show the expected shape of:
- workflow export markdown
- proposal packets
- draft `SKILL.md` scaffolds

Files:
- `example-export.md`
- `example-proposal.md`
- `example-draft.SKILL.md`

These are safe to publish because they do not contain real user session data.
