# Acknowledgments

This repository explicitly acknowledges the public work that served as its base inspiration and starting point:

- [Romanescu11/hermes-skill-factory](https://github.com/Romanescu11/hermes-skill-factory)

## Attribution note

The original repository introduced the core Skill Factory concept for Hermes: observing repeated workflows and turning them into reusable skills.

This repository builds from that base and reworks it into a more conservative Hermes-native implementation focused on:

- current plugin API compatibility
- deterministic hook-visible evidence capture
- explicit export and proposal artifacts
- cleaner public documentation and handoff materials

## Intent

The goal of this acknowledgement is straightforward: we used that work as a conceptual and structural base, and this repository should say so clearly.
